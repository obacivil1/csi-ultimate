منتج توليد العملاء المحتملين (Lead Generation)
═══════════════════════════════════════════

المصادر المتاحة حالياً:

1. Muqawil.org — مقاولين سعوديين (8,136 سجل)
2. Expatriates.com — باحثين عن عمل + إعلانات وظائف
3. Bayt.com — وظائف (محتاج حل Cloudflare)

═══════════════════════════════════════════
الاستخدام:

# مقاولين من الرياض مع إيميل
node muqawil-leads.mjs --city=الرياض --has-email

# مقاولين من جدة مع هاتف
node muqawil-leads.mjs --city=جدة --has-phone

# تصدير CSV لكل المقاولين
node muqawil-leads.mjs --has-email --format=csv

# مقاولين كبار (large)
node muqawil-leads.mjs --size=large

═══════════════════════════════════════════
خطة التوسع:

1. جمع بيانات من:
   - LinkedIn (منشورات توظيف)
   - Bayt.com (وظائف)
   - Indeed (وظائف)
   - Twitter/X (إعلانات وظائف)
   
2. تصدير تقارير شهرية للعملاء
3. CRM متكامل مع نظام العروض
